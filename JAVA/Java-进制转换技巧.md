# 进制转换技巧

## 一、常用进制一览

| 进制 | 基数 | 数字符号 | 前缀标识 | Java 表示 |
|------|------|----------|----------|-----------|
| 二进制 (Binary) | 2 | 0, 1 | `0b` | `0b1010` |
| 八进制 (Octal) | 8 | 0~7 | `0` | `012` (即 10) |
| 十进制 (Decimal) | 10 | 0~9 | — | `10` |
| 十六进制 (Hex) | 16 | 0~9, A~F | `0x` | `0x1A` |

## 二、转十进制（按权展开求和）

**公式**：每一位的值 × 基数^位权，然后相加。

```
二进制 1011 → 十进制
= 1×2³ + 0×2² + 1×2¹ + 1×2⁰
= 8 + 0 + 2 + 1
= 11

十六进制 2A3 → 十进制
= 2×16² + 10×16¹ + 3×16⁰    (A=10)
= 512 + 160 + 3
= 675

八进制 157 → 十进制
= 1×8² + 5×8¹ + 7×8⁰
= 64 + 40 + 7
= 111
```

## 三、十进制转其他进制

### 整数部分：除基取余，逆序排列

```
十进制 13 → 二进制
13 ÷ 2 = 6 …… 1 ↑
 6 ÷ 2 = 3 …… 0 ↑
 3 ÷ 2 = 1 …… 1 ↑
 1 ÷ 2 = 0 …… 1 ↑
结果：1101 (从下往上读)

十进制 300 → 十六进制
300 ÷ 16 = 18 …… 12(C) ↑
 18 ÷ 16 = 1  …… 2      ↑
  1 ÷ 16 = 0  …… 1      ↑
结果：0x12C
```

### 小数部分：乘基取整，顺序排列

```
十进制 0.625 → 二进制
0.625 × 2 = 1.25 → 取 1
0.25  × 2 = 0.50 → 取 0
0.50  × 2 = 1.00 → 取 1
结果：0.101
```

## 四、二/八/十六进制互转（速算）

### 二进制 ↔ 八进制（3位一组）

```
二进制 1 101 110 → 八进制
从右往左，3位一组：
110 → 6
101 → 5
1   → 1（不足3位补零）
结果：0156

八进制 74 → 二进制
7 → 111
4 → 100
结果：0b111100
```

### 二进制 ↔ 十六进制（4位一组）

```
二进制 1010 1110 → 十六进制
1010 → A
1110 → E
结果：0xAE

十六进制 3F → 二进制
3 → 0011
F → 1111
结果：0b111111
```

## 五、编程中的进制转换

### Java 内置方法

```java
// 十进制 → 其他进制（Integer 类）
Integer.toBinaryString(13);   // "1101"
Integer.toOctalString(13);    // "15"
Integer.toHexString(300);     // "12c"

// 其他进制 → 十进制
Integer.parseInt("1101", 2);  // 13
Integer.parseInt("157", 8);   // 111
Integer.parseInt("2A", 16);   // 42

// 任意进制转换（2~36）
Integer.toString(42, 16);     // "2a"
Integer.parseInt("2a", 16);   // 42
```

### 手算口诀

| 转换方向 | 方法 |
|----------|------|
| 任意 → 10 | 按权展开求和 |
| 10 → 任意 | 整数：除基取余逆序；小数：乘基取整顺序 |
| 2 ↔ 8 | 3位一组 |
| 2 ↔ 16 | 4位一组 |
| 8 ↔ 16 | 以二进制为桥梁，先 8→2→16 |

## 六、快速估算：二的幂

在编程中常用，和二进制紧密相关。

| 2ⁿ | 十进制 | 常见用途 |
|-----|--------|----------|
| 2⁸ | 256 | byte 范围，IPv4 段 |
| 2¹⁰ | 1024 | 1KB，`int` 最大值 2³¹-1≈2.1亿 |
| 2¹⁶ | 65536 | short 范围，端口号上限 |
| 2³² | ≈42.9亿 | int 范围，IPv4 总数 |
| 2⁶⁴ | ≈1.8×10¹⁹ | long 范围 |

> 编程中常用的速算：`2^10 ≈ 10^3`（1024 ≈ 1000），所以 `2^20 ≈ 10^6`（百万），`2^30 ≈ 10^9`（十亿）。
