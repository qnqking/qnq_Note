## 学习重点
- 封装：私有属性 `_name` / `__name`，通过 getter/setter 访问
- 继承：`class Child(Parent)`，多继承支持
- 多态：同一个方法，不同对象的行为不同
- `__str__` 控制打印输出，`__eq__` 控制 == 比较
- 抽象类不能被实例化，强制子类实现抽象方法
